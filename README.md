# Data Structures and Algorithms (Course Assignments)

## Overview

This repository contains my implementations of core data structures and algorithms as part of a university-level Data Structures course.

The focus of this repository is on:
- Building fundamental data structures from scratch  
- Understanding algorithmic efficiency and trade-offs  
- Writing clean, modular, and well-structured code  

---

## Topics Covered

### Linear Data Structures
- Arrays  
- Linked Lists (Singly / Doubly)  
- Stacks  
- Queues  

---

### Trees
- Binary Trees  
- Binary Search Trees  
- Tree traversals (inorder, preorder, postorder)  

---

### Graphs
- Graph representations (adjacency list / matrix)  
- Traversals (BFS, DFS)  
- Basic graph algorithms  

---

### Searching and Sorting
- Linear Search  
- Binary Search  
- Sorting algorithms:
  - Bubble Sort  
  - Selection Sort  
  - Insertion Sort  
  - Merge Sort  
  - Quick Sort  

---

### Additional Concepts
- Recursion  
- Time and space complexity analysis  
- Basic problem-solving techniques
